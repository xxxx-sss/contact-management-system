📇 接触管理系统（通讯录管理系统）

这是一个基于 C++ 开发的控制台通讯录管理程序，实现了联系人信息的基础增删改查功能。

---

✨ 功能列表
- 添加联系人
- 显示所有联系人
- 删除指定联系人
- 查找联系人信息
- 修改联系人信息
- 清空所有联系人
- 退出系统

---

📂 项目结构
contact-management-system/
├── 通讯录管理系统/          # 项目文件夹
│   ├── Debug/              # 编译生成文件（可忽略）
│   └── 通讯录管理系统.cpp  # 主程序代码
├── .vs/                    # VS 配置文件（可忽略）
├── 通讯录管理系统.sln       # VS 解决方案文件
└── README.md               # 项目说明文件

---

🚀 运行方式
1. 使用 Visual Studio 打开 通讯录管理系统.sln
2. 右键项目 → 设为启动项目
3. 点击「本地 Windows 调试器」即可运行程序

---

📌 说明
- 联系人信息包括：姓名、性别、年龄、电话、地址
- 系统采用控制台交互，所有操作均通过数字菜单选择
- 适合作为 C++ 基础语法（数组、结构体、函数封装）的练手项目

本项目为学习用途，代码仅供参考。
